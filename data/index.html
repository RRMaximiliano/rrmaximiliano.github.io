Data Projects

I would like to share some datasets and repositories that I have created or cleaned that might be helpful for other researchers.

  • Github repo

R

eleccionesNic

An R package that allows users to use several datasets containing information on the presidential elections in Nicaragua from 1990 to 2006.

  • Github repo

Data repositories

National Police of Nicaragua data repository

Wrote a code that scrape information from officials reports from the National Police of Nicaragua. The extracted data contains information for type of crime, complaints, and solved cases in Nicaragua from 1980 to 2019. First tidy dataset that is available to the public on crime in Nicaragua at the department level.

  • Github repo

Instituto de Medicina Legal data repository

Wrote a code that scrape information from officials reports from the Institute of Legal Medicine. The extracted data contains information for typology of violence and total cases by medical circunscripción.

  • Github repo

Instituto Nicaragüense de Seguridad Social data repository

Wrote a code that scrape information from officials reports from the Nicaraguan Institute of Social Security. The extracted data contains information for the total insured population by gender.

  • Github repo

Coronavirus data repository for Nicaragua

Scraped information from the Observatorio Ciudadano de Nicaragua (Non-official information on the current state of COVID-19 in Nicaragua) that allows users to use the datasets and plot information at the national and department level.

  • Github repo

Nicaraguan rulers

Hand-coded information on all the rulers of Nicaragua. Currently, the data in this repository contains information from the year 1508 to 2020. First tidy dataset that is available to the public on Nicaraguan rulers,

  • Github repo

Nicaraguan population 18th and 19th Century

Hand-code information form archives on the Nicaraguan population in the 18th and 19th Century.

  • Github repo

Political Prisoners (Apr 2018 - Apr 2019)

This dataset contains information about the political prisoners from the protest in Nicaragua since April 2018. The data comes from the “Comité Pro Liberación de Presas y Presos Políticos”. Version updated: April 1, 2019.

  • R
  • Stata
  • CSV
